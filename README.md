Coding challenge application showing Movies currently playing. 

###Tech: 

Ruby
Sinatra
Backbone
Handlebars

###To run locally: 

`gem install sinatra`
`gem install json`
`gem install httparty`

`ruby application.rb`

Visit `http://localhost:4567/` in your browser

###Hosted: 

[Switch Movie App](http://switch-movies.herokuapp.com)
